git checkout -b <branchname> / per creare un branch in locale

git checkout <branchname> / per spostarmi di branch

git branch / lista di tutti i branchs

git branch -d <branchname> / eliminare un branch en locale

git push origin <branchname> / manda il branch dentro al server git 

git merge <branchname> / mette tutta l'informazione del branch selezionato nel branch in cui sei posizionato

git push origin --delete <branch> / eliminare il branch in git 
git push origin :<branchname>     / eliminare il branch in git


git commit -m "" / commento il salvataggio
git add --all / salvare le modifiche 


git fetch --all / Una mossa avanzata che consente di recuperare tutti i repository remoti registrati e i relativi branch

git fetch <remote> <branch> / Come il comando descritto sopra, a eccezione del fatto che recupera solo il branch specificato.

git fetch --all / Una mossa avanzata che consente di recuperare tutti i repository remoti registrati e i relativi branch
